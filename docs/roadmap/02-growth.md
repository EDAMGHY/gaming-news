# Phase 2: Growth

## Goal

After the MVP works, improve content discovery, editorial depth, and reader engagement.

## 1. Taxonomy And Discovery

- [ ] Add category pages like `/categories/[slug]`.
- [ ] Add genre pages like `/genres/[slug]`.
- [ ] Add platform pages like `/platforms/pc`, `/platforms/ps5`, `/platforms/switch-2`.
- [ ] Add narrative tag pages if useful for discovery.
- [ ] Add filters on `/games` for platform, genre, release window, and game length.
- [ ] Add filters on `/reviews` for rating, platform, and genre.

## 2. Editorial Features

- [ ] Add Guides as either a category under articles or a separate collection later.
- [ ] Add Features or Opinion as article categories.
- [ ] Add evergreen list content like `Best RPGs`, `Best PS5 Games`, and `Upcoming Indie Games`.
- [ ] Add related articles to review pages.
- [ ] Add related reviews/articles to game pages.
- [ ] Add author profile pages.
- [ ] Add editorial policy page.
- [ ] Add review scoring policy page.

## 3. Homepage Improvements

- [ ] Add Latest News block that automatically queries recent articles.
- [ ] Add Browse by Platform block.
- [ ] Add Browse by Genre block.
- [ ] Add Editor's Picks block.
- [ ] Add Most Anticipated Games block.
- [ ] Add newsletter signup block.

## 4. User Engagement

- [ ] Add newsletter signup integration.
- [ ] Add share buttons on articles and reviews.
- [ ] Add comments when moderation strategy is clear.
- [ ] Add reaction buttons only if they support product goals.
- [ ] Add reading time to articles.
- [ ] Add review summary box near the top of review pages.

## 5. Content Operations

- [ ] Add stronger admin list columns for articles, reviews, and games.
- [ ] Add editorial statuses if needed beyond Payload drafts.
- [ ] Add internal notes fields for editors.
- [ ] Add source/citation fields for news articles if needed.
- [ ] Add embargo date or review copy disclosure fields if needed.

## 6. Performance And UX

- [ ] Audit mobile homepage layout.
- [ ] Audit card image sizes and loading behavior.
- [ ] Add empty states for archive pages.
- [ ] Add better loading and error states where needed.
- [ ] Improve archive page sorting and pagination UX.

## 7. Analytics

- [ ] Add privacy-friendly analytics.
- [ ] Track popular articles.
- [ ] Track popular reviews.
- [ ] Track game page traffic.
- [ ] Use analytics to decide future content categories.
